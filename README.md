# Project Overview

This project is organized for a simple data workflow:

- `data/`: raw or reference data
- `notebooks/`: analysis and experimentation
- `src/`: reusable Python code
- `results/`: exported outputs and evaluation results

## Structure

```text
.
├── data/
│   └── README.md
├── notebooks/
│   ├── 01_EDA.ipynb
│   ├── 02_feature_engineering.ipynb
│   └── 03_modeling.ipynb
├── src/
│   ├── preprocess.py
│   ├── features.py
│   └── train.py
├── results/
│   └── model_performance.csv
├── requirements.txt
└── README.md
```
